A cool website for Piper Grace with games, a 3D horse creator, and more!

## Features

- **Horse Creator** - Design your own custom horse with different colors
- **Horse Runner** - Endless runner game with your custom horse
- **Virtual Stable** - Take care of your horse (Tamagotchi-style)
- **Art Studio** - Draw and create masterpieces
- **Puzzle Master** - Solve photo puzzles
- **Memory Matrix** - Test your memory

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS v4 + DaisyUI (Synthwave theme)
- React Three Fiber (3D)
- Framer Motion (Animations)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.
